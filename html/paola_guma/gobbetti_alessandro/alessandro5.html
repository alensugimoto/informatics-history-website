Interesting things about Linux

Linux Name

Linus Torvalds wanted to call Linux “Freax”, a portmanteau of "free", "freak", and "x". He considered the name “Linux”, but he refused it as too egotistical.
Ari Lemmke, an volunteer administrator of the server where the project was stored, did not think that "Freax" was a good name, so he renamed it "Linux" on the server without consulting Torvalds. Afterward, obviously, Torvalds approved "Linux".

Linux Mascot

In 1996 Torvalds announced that Linux would have a penguin, called Tux, as mascot. In an interview Linus commented on the penguin bite:

I've been to Australia several times, these days mostly for Linux.Conf.Au. But my first trip—and the one when I was bitten by a ferocious fairy penguin: you really should keep those things locked up!—was in 93 or so, talking about Linux for the Australian Unix Users Group.

Now Tux is the most commonly used icon for Linux, although different Linux distributions depict Tux in various styles.

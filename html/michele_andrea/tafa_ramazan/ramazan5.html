Bi-Directional Input

Around 1978 Elbit Systems in Israel developed a CP/M capable microcomputer named the DS2100. They made an agreement with MicroPro to develop a version of WordStar that supported both English and Hebrew input. The concept was ground braking, as Hebrew is written right-to-left and all word processors at the time assumed left-to-right. WordStar, as developed by Elbit, was the first word processor that offered bi-directional input and mixed alphabets.

File Extensions

The file extension for Windows files of WordStar is .WSD
The file extension for Windows templates of WordStar is .WST
The file extension for Windows macros of WordStar is .WMC
The file extension for for Windows temporary files of WordStar is !WS
WordStar 2000 for DOS and UNIX PC did not have a fixed extension but .DOC and .WS2 were common
